Как получить API - Ключ:

Для аналитики ваших продаж на WildBerries необходимо подключить кабинет поставщика по API - ключу. На портале поставщика перейдите в раздел «Настройки»

  1. Нажиме «Доступ к API» (https://suppliers-portal.wildberries.ru/supplier-settings/access-to-api)
  2. Сгенерируйте API-ключ
  3. Скопируйте API-ключ в формате Base64
  4. Введите ваш ключ в сервисе в разделе «Кабинет продавца» → «Магазины» и нажмите «Сохранить».
